# register-new-page-in-index

После создания любой новой страницы верстки добавляй ссылку на нее в `app/index.html`.

## Source of truth

- Файл-реестр страниц: `app/index.html`
- Блок ссылок: список внутри `<ul class="fs-18">`

## Задача

1. Найди список страниц в `app/index.html`.
2. Добавь новый `<li>` со ссылкой на созданную страницу.
3. Держи единый формат ссылки:
   - текст: понятное название страницы
   - в скобках: точное имя файла, например `(new-page.html)`
4. Проверь, что `href` совпадает с фактическим именем HTML-файла.
5. Убедись, что `<title>` целевой страницы совпадает с названием страницы в этой записи реестра.
6. Не удаляй и не переименовывай существующие пункты без отдельного запроса.

## Формат элемента

```html
<li>
    <a href="new-page.html">Название страницы (new-page.html)</a>
</li>
```

## Проверки перед завершением

- В `app/index.html` присутствует ссылка на новую страницу.
- Имя файла в `href` совпадает с именем в тексте ссылки.
- `<title>` страницы совпадает с зарегистрированным названием в `app/index.html`.
- Разметка списка остается валидной (`<ul>/<li>/<a>`).

## Критерий приемки

- Каждая новая страница проекта зарегистрирована в `app/index.html` в рамках той же задачи.

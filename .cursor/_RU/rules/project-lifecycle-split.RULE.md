# Политика разделения жизненного цикла проекта

- Используй `commands/init-layout-project.md` для setup и governance-решений.
- Используй `commands/develop-layout-task.md` для ежедневных задач реализации.
- Не смешивай нерешенные вопросы инициации с этапом разработки.
- Если инициация не завершена, заверши или уточни ее до старта реализации.
